Arbori (Structuri Arborescente)

Tot ce trebuie să știi pentru BAC: Concepte, Vectorul de Tați și Arbori Binari

1. Ce este un Arbore cu Rădăcină?

Un arbore cu rădăcină este un caz particular de graf neorientat, conex și aciclic, în care un nod special este desemnat ca fiind rădăcină. Rădăcina stabilește o ierarhie clară de tip „părinte-copil” în întreaga structură.

Vocabularul obligatoriu pentru examen:

  • Rădăcină: Nodul situat pe cel mai înalt nivel ierarhic (singurul nod care nu are un părinte).
  • Tată (Părinte): Dacă există o muchie între nodul u și nodul v, iar u se află pe un nivel mai apropiat de rădăcină, atunci u = tatăl lui v.
  • Fiu (Copil): În exemplul anterior, v = fiul lui u.
  • Frunză (Nod terminal): Un nod care **nu are niciun fiu** (gradul său în ierarhie este 0).
  • Frați: Nodurile care au același tată.
  • Ascendenți (Strămoși): Toate nodurile aflate pe drumul unic de la un nod dat până la rădăcină.
  • Descendenți: Toate nodurile care se află în subarborii generați de fiii unui nod dat.
  • Înălțimea arborelui: Numărul maxim de muchii de la rădăcină până la cea mai îndepărtată frunză (sau numărul maxim de niveluri minus 1).

2. Reprezentarea în Memorie. Vectorul de Tați (T)

La BAC, cea mai importantă metodă de reprezentare a unui arbore cu rădăcină este Vectorul de Tați (notat de regulă cu T).

Este un vector cu n elemente (unde n este numărul de noduri), definit astfel:
• T[i] = tatăl nodului i
• T[i] = 0, dacă nodul i este chiar rădăcina arborelui.

Exemplu practic: Să presupunem că avem un arbore cu 7 noduri, unde rădăcina este nodul 4. Structura lui este reprezentată de următorul vector de tați:

Nodul (i) 1 2 3 4 5 6 7
T[i] 4 1 1 0 4 5 5

3. Cum rezolvi rapid întrebările grilă cu Vectorul de Tați

Pe baza tabelului de mai sus, la Subiectul I se cer instantaneu următoarele proprietăți. Iată algoritmii logici rapizi:

  • Cum găsești rădăcina?
    Cauți poziția pe care se află valoarea 0. În tabel, T[4] = 0, deci rădăcina este 4.
  • Cum găsești frunzele (nodurile terminale)?
    Frunzele sunt nodurile care **NU apar deloc ca tați** în vectorul T.
    Analiză: Valorile din vectorul T sunt: 4, 1, 0, 5. Numerele de la 1 la 7 care lipsesc cu desăvârșire sunt: 2, 3, 6, 7. Acestea sunt frunzele arborelui!
  • Cum găsești fiii unui nod x?
    Sunt toate indicile i pentru care T[i] == x.
    De exemplu, cine sunt fiii lui 4? Căutam unde apare valoarea 4 în vector. Apare pe pozițiile 1 și 5. Deci fiii lui 4 sunt 1 și 5.
  • Cum determini numărul de muchii?
    Orice arbore cu n noduri are exact n - 1 muchii. Pentru exemplul nostru cu 7 noduri, numărul de muchii este mereu 6.

4. Arbori Binari (Structură specială)

Un arbore binar este un arbore cu rădăcină în care **fiecare nod poate avea maximum doi fii**: un fiu stâng și un fiu drept. Un nod poate avea 0, 1 sau 2 fii.

Metode de Parcurgere a Arborilor Binari (Subiect Clasificabil):

Parcurgerea reprezintă vizitarea fiecărui nod o singură dată. Există 3 strategii fundamentale bazate pe poziția Rădăcinii față de Subarborele Stâng (S) și Subarborele Drept (D):

  1. Preordine (RSD - Rădăcină, Stânga, Dreapta): Se vizitează mai întâi rădăcina, apoi se parcurge recursiv subarborele stâng, apoi cel drept.
  2. Inordine (SRD - Stânga, Rădăcină, Dreapta): Se parcurge întâi subarborele stâng, se vizitează rădăcina, iar la final se parcurge subarborele drept.
  3. Postordine (SDR - Stânga, Dreapta, Rădăcină): Se parcurge mai întâi subarborele stâng, apoi cel drept, iar rădăcina este vizitată ultima.

Exemplu rapid de parcurgere: Pentru un arbore cu rădăcina 1, fiul stâng 2 (care are la rândul lui fiul stâng 4) și fiul drept 3:

  • Preordine (RSD): 1, 2, 4, 3
  • Inordine (SRD): 4, 2, 1, 3
  • Postordine (SDR): 4, 2, 3, 1

5. Cum se declară un Arbore Binar în C++

Dacă ai nevoie de o alocare dinamică la Subiectul al III-lea, se folosesc structurile cu pointeri către nodul stâng și drept:

struct Nod {
    int info;
    Nod *stâng;  // Pointer către fiul stâng
    Nod *drept;  // Pointer către fiul drept
};

// Implementarea recursivă a parcurgerii în INORDINE (SRD)
void inordine(Nod *r) {
    if (r != NULL) {
        inordine(r->stâng);  // S
        cout << r->info << " ";  // R
        inordine(r->drept);  // D
    }
}

Sfat pentru examenul de BAC

Când primești la Subiectul I un exercițiu cu un vector de tați, cel mai sigur mod de a nu greși este să desenezi arborele pe ciornă. Porniți de la nodul care are valoarea 0 (rădăcina), apoi căutați în tabel nodurile care îl au ca tată, desenându-le dedubtul său, nivel cu nivel. Vizualizarea grafică te va ajuta să răspunzi corect la orice întrebare legată de frunze, înălțime sau număr de descendenți.
