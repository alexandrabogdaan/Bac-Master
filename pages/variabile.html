Variabile și tipuri de date

Noțiuni fundamentale despre memorie, valori și reprezentarea datelor în calculator

Definiția variabilei

O variabilă este o zonă de memorie identificată printr-un nume, în care se stochează o valoare ce poate fi modificată în timpul execuției programului.

Numele variabilei rămâne constant, dar valoarea se poate schimba.

Memoria și reprezentarea în biți

Calculatorul stochează toate datele sub formă de biți (0 și 1). Un bit reprezintă cea mai mică unitate de informație.

8 biți formează 1 byte (octet).

  • 1 byte = 8 biți
  • 1 KB = 1024 bytes
  • 1 MB = 1024 KB

Fiecare tip de dată ocupă un număr diferit de bytes în memorie.

Tipuri de date și memorie

Fiecare tip de dată este stocat diferit în memorie:

  • int – 4 bytes (32 biți)
  • float – 4 bytes (32 biți)
  • double – 8 bytes (64 biți)
  • char – 1 byte (8 biți)
  • bool – 1 byte (în majoritatea compilatoarelor)

Această alocare permite calculatorului să reprezinte valori diferite în format binar.

Tipul int (numere întregi)

Tipul int este reprezentat pe 32 de biți.

Un bit este folosit pentru semn (pozitiv/negativ), iar restul pentru valoare.

Interval aproximativ: 

-2,147,483,648 până la 2,147,483,647

Tipuri reale (float și double)

Numerele reale sunt stocate în format cu virgulă mobilă (floating point).

  • float – 32 biți (precizie mai mică)
  • double – 64 biți (precizie mai mare)

Aceste tipuri nu stochează exact toate numerele reale, ci o aproximație.

Tipul char

Un caracter este stocat pe 8 biți (1 byte). Fiecare caracter are un cod numeric (ASCII). 

'A' = 65
'a' = 97

Astfel, caracterele sunt de fapt numere în memorie.

Tipul bool

Tipul boolean poate avea doar două valori:

  • true (1)
  • false (0)

Este folosit în condiții și decizii logice.

Declararea variabilelor

O variabilă trebuie declarată înainte de utilizare. 

int a = 10;
float x = 3.5;
char c = 'A';
bool ok = true;

Concluzie

Variabilele sunt baza oricărui program. Ele permit stocarea și manipularea datelor în memorie, iar tipul lor determină modul de reprezentare în biți.
