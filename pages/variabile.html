Variabile și tipuri de date

Noțiuni fundamentale despre memorie, valori și reprezentarea datelor în calculator

Definiția variabilei

În informatică, gestionarea corectă a datelor pornește de la înțelegerea conceptului de stocare temporară.

Concept fundamental: O variabilă este o zonă de memorie identificată printr-un nume, în care se stochează o valoare ce poate fi modificată în timpul execuției programului.

În timp ce numele variabilei (identificatorul) rămâne constant, conținutul acesteia se poate schimba de oricâte ori este necesar.

Memoria și reprezentarea în biți

Calculatorul nu înțelege direct numerele zecimale sau literele, ci stochează toate datele sub formă de biți (stări de 0 și 1).

Bitul

Reprezintă cea mai mică unitate de informație dintr-un sistem de calcul, având valoarea 0 sau 1.

Octetul (Byte)

Este unitatea principală de măsură. Un grup de 8 biți formează 1 byte, capabil să stocheze o valoare între 0 și 255 (sau -128 și 127).

Grila de transformare a unităților:
  • 1 Byte = 8 biți
  • 1 KB (Kilobyte) = 1024 Bytes
  • 1 MB (Megabyte) = 1024 KB

Tipuri de date și alocarea în memorie

Fiecare variabilă are nevoie de un anumit spațiu pentru a-și reprezenta valorile în format binar. Structura completă a tipurilor primitive din C++ este următoarea:

Tip de dată Dimensiune memorie Tipul de valori stocate
int 4 bytes (32 biți) Numere întregi (cu sau fără semn)
float 4 bytes (32 biți) Numere reale (cu virgulă, precizie simplă)
double 8 bytes (64 biți) Numere reale (precizie dublă)
char 1 byte (8 biți) Un singur caracter codificat ASCII
bool 1 byte Valori logice (adevărat sau fals)

Analiza detaliată a tipurilor de date

Pentru examenul de Bacalaureat, este esențial să înțelegi cum funcționează limitările fiecărui tip în parte:

Tipul int (Numere întregi)

Fiind reprezentat pe 32 de biți, primul bit este utilizat pentru a salva semnul (0 pentru pozitiv, 1 pentru negativ), iar restul de 31 de biți salvează valoarea efectivă.

Intervalul de valori permise:

-2,147,483,648 până la 2,147,483,647

Tipurile reale (float și double)

Numerele reale sunt salvate în formatul standard cu virgulă mobilă (floating point).

Important: Aceste tipuri nu stochează întotdeauna valoarea matematică exactă, ci o aproximație extrem de fidelă, generând uneori mici erori de precizie la zecimalele îndepărtate.

Tipul char (Caractere)

Deși reprezintă o literă sau un simbol, în memorie tipul char este salvat sub forma unui număr întreg cuprins între 0 și 255, reprezentând codul ASCII al acelui caracter.

'A' = 65  |  'a' = 97  |  '0' = 48

Tipul bool (Valori logice)

Folosit exclusiv în structurile condiționale și decizionale pentru a reține starea unei instrucțiuni. Acceptă doar două stări:

  • true (reprezentat intern prin valoarea 1 sau orice valoare nenulă)
  • false (reprezentat intern prin valoarea 0)

Declararea variabilelor în cod

Înainte de a putea introduce date sau efectua calcule într-o variabilă, compilatorul C++ te obligă să o declari, specificându-i clar tipul:

int a = 10;       // Alocă 4 bytes pentru un număr întreg
float x = 3.5;    // Alocă 4 bytes pentru un număr real
char c = 'A';     // Alocă 1 byte pentru un caracter (cod ASCII 65)
bool ok = true;   // Alocă 1 byte pentru un indicator logic
Concluzie:
Variabilele constituie fundamentul programării structurate. Alegerea corectă a tipului de dată este vitală pentru a asigura atât utilizarea eficientă a memoriei RAM, cât și prevenirea erorilor de depășire de tip (overflow) la examenele scrise.
