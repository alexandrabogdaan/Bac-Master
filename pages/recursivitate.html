Recursivitate

Noțiuni fundamentale despre recursivitate în programare

Definiția recursivității

Recursivitatea este o tehnică de programare în care un subprogram se apelează pe sine însuși.

Fiecare funcție recursivă trebuie să aibă un caz de bază.

Exemple

  • Factorialul unui număr
  • Ridicarea la putere a unui număr
  • Șirul lui Fibonacci

Observație

Fără un caz de bază, funcția recursivă nu se va termina niciodată și va duce la o eroare de execuție.

Exemplu în C++

Scriem o funcție C++ care returnează factorialul unui număr natural transmis ca parametru. Varianta nerecursivă (iterativă) este următoarea: 

int factorial(int n)
    {
    if(n == 0)
        return 1; //cazul de bază
    else
        return n * factorial(n-1);
    }
