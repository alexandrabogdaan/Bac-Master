Subprograme

Noțiuni fundamentale despre declararea și utilizarea subprogramelor

Definiția subprogramului

Un subprogram este o colecție de tipuri de date, variabile, instrucțiuni care îndeplinesc o anumită sarcină (calcule, citiri, afișări), atunci când este apelat (folosit) de un program sau de un alt subprogram.

Un subprogram se declară în C++ astfel:

tipulFunctiei numeleFunctiei(parametrii);

Exemple

  • Ex1: int suma(int a, int b);
  • Ex2: bool estePar(int n);

Observație!

O funcție trebuie și definită, nu doar declarată!

Definirea unei funcții se face astfel: 

int suma(int a, int b);
{
      return a + b;
}

Folosirea acestora

Exemplu de implementare:

Atenție: eroare de compilare

În acest caz, vom avea o eroare de compilare:

#include <iostream>
using namespace std;

int main()
        {
        suma(5,10);
        return 0;
        }

int suma(int a, int b)
        {
        return a + b;
        }

Acest lucru este pentru că noi doar am definit funcția după ce am apelat-o, deci compilatorul nu știe ce este suma(5,10) și va da eroare.

Cazul corect

#include <iostream>
using namespace std;

int suma(int a, int b);

int main()
        {
        suma(5,10);
        return 0;
        }

int suma(int a, int b)
        {
        return a + b;
        }

Stocarea funcțiilor în memorie

Funcțiile folosesc memoria de pe Stiva (stack)

Anatomia unei funcții

Antetul acesteia este:

tipulFunctiei numeleFunctiei(parametrii);
ex. int suma (int a, int b)

Corpul unei funcții este:

{
    return a+b;
}

Când folosim o funcție într-un program, spunem: apelăm funcția:

int main()
        {
         cout<< suma(5,10); // Afiseaza 15
        }

Particularități

Funcții fără valoare returnată

Funcțiile care nu returnează o valoare au tipul de returnare: void

void afisarePar(int a)
{
    if(a % 2 == 0)
        cout << a << ' ';   
} //Afișează numărul dacă este par

Transmiterea prin referință

Atunci când dorim ca o funcție să poată modifica direct valoarea unei variabile din afara ei (din main sau din altă funcție), folosim transmiterea prin referință.

Diferența față de transmiterea prin valoare

Spre deosebire de transmiterea prin valoare (unde funcția primește doar o copie a datelor), prin referință funcția lucrează direct cu variabila originală. Adăugăm simbolul & (amperand) înaintea numelui parametrului. 

void dubleaza(int &x) // x este transmis prin referință
    {
    x = x * 2; // Modificarea afectează direct variabila din apel
    }

int main()
    {   
    int numar = 10;
    dubleaza(numar);
    cout << numar; // Va afișa 20, deoarece 'numar' a fost modificat direct!
    }
