Metoda Backtracking

Algoritmi de căutare sistematică a tuturor soluțiilor posibile.

Ce este backtracking?

Backtracking este o metodă de rezolvare a problemelor care constă în generarea tuturor soluțiilor posibile, construite pas cu pas, și eliminarea celor care nu respectă condițiile problemei.

Algoritmul explorează toate variantele posibile, dar renunță (face „backtrack”) atunci când o soluție parțială devine invalidă.

Ideea algoritmului

Backtracking construiește soluția treptat, alegând pe rând valori dintr-un set. Dacă la un moment dat o alegere nu este validă, algoritmul revine la pasul anterior și încearcă altă variantă.

Este similar cu explorarea unui arbore de soluții.

Aplicații ale backtracking-ului

  • generarea permutărilor
  • problema reginelor (N-Queens)
  • aranjamente și combinații
  • probleme de tip sudoku
  • probleme de parcurgere a grafurilor

Schemă generală Backtracking

Majoritatea problemelor de backtracking pot fi rezolvate folosind o structură recursivă generală. 

int x[100];

void back(int k)
{
    for(i=1;i<=n;i++)
    {
        x[k]=i;

        if(valid(k))
        {
            if(solutie(k)) afis(k);
            else back(k+1);
        }
    }
}

Unde:

  • valid(k) verifică dacă alegerea este corectă
  • solutie(k) verifică dacă am ajuns la o soluție completă

Generarea permutărilor

Permutările unui set de n elemente reprezintă toate ordonările posibile ale acestora. 

int x[10], f[10], n=3;

void back(int k)
{
    for(i=1;i<=n;i++)
    {
        if(f[i]==0)
        {
            x[k]=i;
            f[i]=1;

            if(k==n) afis(k);
            else back(k+1);

            f[i]=0;
        }
    }
}

Vectorul f marchează elementele deja folosite.

Probleme tip BAC

În problemele de backtracking de la BAC, cel mai des se cer:

  • generarea tuturor permutărilor
  • generarea combinațiilor
  • aranjamente cu condiții
  • numere cu cifre date

Ideea este mereu aceeași: construim soluția pas cu pas și verificăm condițiile.

Schema de reținut - Backtracking

1. Ideea de bază:

Construim soluția pas cu pas și revenim dacă o alegere nu este bună.

2. Structura generală:

void back(int k)
{
    for(i=1;i<=n;i++)
    {
        x[k]=i;

        if(valid(k))
        {
            if(solutie(k)) afis(k);
            else back(k+1);
        }
    }
}

3. Tipuri importante:

  • Permutări → toate elementele o singură dată
  • Combinații → alegere fără ordine
  • Aranjamente → alegere cu ordine

4. Ideea-cheie:

Backtracking = încercare + verificare + revenire

Generarea combinațiilor

int x[10], n=5, k=3;
void back(int p)
{
    for(i=x[p-1]+1;i<=n;i++)
    {
        x[p]=i;

        if(p==k) afis(p);
        else back(p+1);
    }
}

Se generează combinațiile de k elemente dintr-un set de n.
