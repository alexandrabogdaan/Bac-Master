Metoda Backtracking

Teorie + probleme BAC rezolvate corect

Definiție

Metoda backtracking este o metodă de parcurgere sistematică a tuturor soluțiilor posibile ale unei probleme, construind soluția pas cu pas și renunțând la variantele care nu respectă condițiile impuse.

Ideea metodei

Se construiește un vector soluție x[] = (x[1], x[2], ..., x[n]), unde fiecare element x[k] ia valori dintr-o mulțime de valori posibile.

La fiecare pas: alegem o valoare → verificăm → continuăm sau revenim.

Schemă generală

void back(int k)
{
    for(i=A;i<=B;i++)
    {
        x[k]=i;

        if(ok(k))
        {
            if(solutie(k)) afis(k);
            else back(k+1);
        }
    }
}

Termeni importanți

  • Condiții externe: valorile posibile pentru x[k]
  • Condiții interne: verificarea validității parțiale
  • Soluție finală: vector complet corect

Schema de reținut

Backtracking = alegere → verificare → revenire

Exercițiul 1 (BAC 2025)

Enunț:

Utilizând metoda backtracking se generează toate permutările elementelor mulțimii ordonate {1, 2, 3, 4, 5, 6}; pentru fiecare permutare, pe primele trei poziții sunt doar valori pare, iar pe ultimele trei poziții sunt doar valori impare. Primele șase soluții generate sunt: (2,4,6,1,3,5), (2,4,6,1,5,3), (2,4,6,3,1,5), (2,4,6,3,5,1), (2,4,6,5,1,3), (2,4,6,5,3,1). Indicați a șaptea soluție generată.

Rezolvare:

Primele trei poziții rămân fixe (2,4,6), iar ultimele 3 sunt permutări ale numerelor impare. După epuizarea permutărilor impare, algoritmul modifică prefixul.

Răspuns: (4,2,6,1,3,5)

Exercițiul 2 (BAC 2024)

Enunț:

Utilizând metoda backtracking se generează toate numerele formate din cifre distincte, cifre a căror sumă este 6. Primele patru soluții generate sunt, în această ordine: 1023, 1032, 105, 1203. Indicați cea de a șasea soluție generată.

Rezolvare:

Se generează numerele în ordine lexicografică a construcției backtracking. După 1203 continuă ramurile valide până la următoarea soluție.

Răspuns: 123

Exercițiul 3 (BAC 2021)

Enunț:

Se generează toate numerele de 3 cifre formate doar din cifre impare (1, 3, 5, 7, 9), utilizând metoda backtracking. Primele soluții generate sunt în ordine de parcurgere. Se cere a șasea soluție generată.

Rezolvare:

Se construiesc toate combinațiile de 3 cifre din mulțimea {1,3,5,7,9}, în ordine crescătoare. Se numără soluțiile până la poziția cerută.

Răspuns: 135

Concluzie

La BAC, backtracking nu se rezolvă prin implementare completă, ci prin analiza ordinii de generare a soluțiilor și înțelegerea restricțiilor.
