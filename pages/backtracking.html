Metoda Backtracking

Teorie + aplicații pentru examenul de Bacalaureat

Ce este backtracking?

Backtracking este o metodă de rezolvare a problemelor în care se construiește soluția pas cu pas, iar dacă o alegere nu este validă, se revine la pasul anterior și se încearcă altă variantă.

Metoda este folosită pentru generarea tuturor soluțiilor posibile ale unei probleme.

Ideea de bază

Soluția unei probleme este reprezentată sub forma unui vector: x[] = (x[1], x[2], ..., x[n]).

Fiecare element x[k] ia valori dintr-o mulțime de valori posibile.

Schemă generală

void back(int k)
{
    for(i=A;i<=B;i++)
    {
        x[k]=i;

        if(ok(k))
        {
            if(solutie(k)) afis(k);
            else back(k+1);
        }
    }
}

Noțiuni importante

  • Condiții externe: valorile posibile pentru x[k]
  • Condiții interne: validarea alegerii curente
  • Soluție finală: vector complet valid

Regula de bază

Backtracking = alegere + verificare + revenire.

Aplicații la Bacalaureat

La BAC, problemele de backtracking NU cer rularea completă a algoritmului, ci doar înțelegerea ordinii de generare a soluțiilor.

Schema de rezolvare la BAC

  • 1. Identifici tipul problemei (permutări / numere / combinări)
  • 2. Observi mulțimea de valori
  • 3. Identifici restricțiile (par/impar, poziții fixe, sumă etc.)
  • 4. Determini ordinea de generare
  • 5. Cauți soluția cerută (a k-a / ultima)

Regulă importantă

NU se construiește complet arborele backtracking. Se urmărește doar ordinea de apariție a soluțiilor.

Exercițiul 1 (BAC 2025)

Se generează permutări ale mulțimii {1,2,3,4,5,6}, astfel încât: primele 3 poziții sunt pare, ultimele 3 sunt impare. 

(2,4,6,1,3,5)
(2,4,6,1,5,3)
(2,4,6,3,1,5)
(2,4,6,3,5,1)
(2,4,6,5,1,3)
(2,4,6,5,3,1)

A 7-a permutare este: (4,2,6,1,3,5)

Exercițiul 2 (BAC 2024)

Se generează numere cu cifre distincte astfel încât suma cifrelor să fie 6. 

1023
1032
105
1203

A 6-a soluție este: 123

Exercițiul 3 (BAC 2021)

Se generează numere cu cifre distincte astfel încât suma cifrelor să fie 6. 

1023
1032
105
1203

A 6-a soluție este: 132

Concluzie

Backtracking la BAC nu înseamnă implementare completă, ci înțelegerea ordinii de generare a soluțiilor.

Cheia: tip problemă + restricții + ordine lexicografică.
