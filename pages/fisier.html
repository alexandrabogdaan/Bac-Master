Lucrul cu Fișiere

Citirea și scrierea în fișiere text, folosind biblioteca <fstream>

De ce folosim fișiere?

Până acum, datele erau citite de la tastatură (cin) și afișate pe ecran (cout). Atunci când volumul de date este mare sau dorim ca rezultatele să rămână salvate permanent, folosim fișiere: citim datele de intrare dintr-un fișier și scriem rezultatele într-un alt fișier (sau în același).

Observație: Pentru a lucra cu fișiere în C++, trebuie inclusă biblioteca <fstream>. Aceasta pune la dispoziție trei tipuri de obiecte: ifstream (citire), ofstream (scriere) și fstream (citire și scriere).

Tipurile de fluxuri (streams)

ifstream

Folosit pentru citirea datelor dintr-un fișier (input file stream).

ofstream

Folosit pentru scrierea datelor într-un fișier (output file stream).

fstream

Permite atât citirea, cât și scrierea în același fișier.

Deschiderea și închiderea unui fișier

Un fișier se deschide fie prin declarare directă (constructor), fie prin metoda open(). La finalul lucrului cu fișierul, acesta trebuie închis cu metoda close(), pentru a salva corect datele și a elibera resursele. 

#include <fstream>
#include <iostream>
using namespace std;

int main()
{
    ifstream fin("date.in");   // deschidere pentru citire
    ofstream fout("date.out"); // deschidere pentru scriere

    int x;
    fin >> x;          // citim un numar din fisier
    fout << x * 2;      // scriem rezultatul in alt fisier

    fin.close();
    fout.close();

    return 0;
}
Capcană frecventă: Dacă fișierul indicat pentru citire (ifstream) nu există, programul nu va da eroare la compilare, dar fluxul va fi invalid, iar citirile vor returna valori eronate sau implicite. Verifică mereu cu fin.is_open() sau if (!fin) dacă fișierul a fost deschis cu succes.

Verificarea deschiderii unui fișier

Este o bună practică să verificăm dacă fișierul a fost deschis corect, înainte de a citi din el:

ifstream fin("date.in");

if (!fin.is_open())
{
    cout << "Eroare: fisierul nu a putut fi deschis!";
    return 1;
}

// altfel, putem citi in siguranta din fin

Citirea datelor dintr-un fișier

Citirea din fișier folosește aceeași sintaxă ca cin, dar cu obiectul ifstream declarat. Putem citi numere, cuvinte (separate de spații) sau linii întregi.

Citirea numerelor / cuvintelor cu >>

Citește valori separate prin spații sau salturi de linie, exact ca la cin.

ifstream fin("numere.in");

int a, b, c;
fin >> a >> b >> c;

cout << a + b + c;

Citirea unei linii întregi cu getline()

Citește toate caracterele de pe o linie, inclusiv spațiile, până la caracterul de linie nouă.

#include <fstream>
#include <string>
using namespace std;

ifstream fin("text.in");
string linie;

getline(fin, linie); // citeste o linie completa, ca string
cout << linie;
Observație: Pentru a citi toate valorile dintr-un fișier, fără a cunoaște numărul lor exact, se folosește citirea în buclă while, care se oprește automat la finalul fișierului (End Of File): 
ifstream fin("numere.in");
int x, suma = 0;

while (fin >> x)
{
    suma += x; // se executa pentru fiecare numar citit, pana la finalul fisierului
}

cout << suma;

Scrierea datelor într-un fișier

Scrierea folosește aceeași sintaxă ca cout, dar cu obiectul ofstream. 

ofstream fout("rezultate.out");

int suma = 42;
fout << "Suma este: " << suma << endl;
fout << "Linia a doua din fisier.";
Capcană frecventă: Dacă fișierul pentru scriere (ofstream) există deja, deschiderea lui implicită îi va șterge complet conținutul anterior (este suprascris). Pentru a adăuga date la finalul unui fișier existent, fără a-l șterge, se folosește modul de deschidere ios::app: 
ofstream fout("rezultate.out", ios::app); // app = append (adaugare la final)

fout << "Aceasta linie se adauga, fara sa stearga ce era inainte.";

Moduri de deschidere a unui fișier

Mod Semnificație
ios::inDeschidere pentru citire (implicit la ifstream)
ios::outDeschidere pentru scriere (implicit la ofstream); șterge conținutul existent
ios::appScriere la finalul fișierului, fără a șterge conținutul existent
ios::truncGolește complet fișierul la deschidere (este implicit pentru ios::out)
ios::binaryDeschidere în mod binar, nu text

Exemplu complet

Programul de mai jos citește un șir de numere întregi dintr-un fișier date.in (numărul nefiind cunoscut dinainte) și scrie în fișierul date.out suma și media lor. 

#include <fstream>
using namespace std;

int main()
{
    ifstream fin("date.in");
    ofstream fout("date.out");

    int x, suma = 0, n = 0;

    while (fin >> x)
    {
        suma += x;
        n++;
    }

    fout << "Suma: " << suma << endl;
    if (n > 0)
        fout << "Media: " << (double)suma / n << endl;

    fin.close();
    fout.close();

    return 0;
}
