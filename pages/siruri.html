Șiruri de caractere

Noțiuni fundamentale despre șirurile de caractere în C++.

Despre șirurile de caractere

În C++, șirurile de caractere sunt reprezentate ca tablouri unidimensionale de elemente de tip char. Un șir de caractere este o succesiune de caractere memorată într-un tablou de tip char, terminată obligatoriu cu caracterul nul '\0'.

Caracterul '\0' are codul ASCII 0 și marchează sfârșitul șirului. Funcțiile din biblioteca <cstring> utilizează acest caracter pentru a determina unde se termină șirul.

Declararea unui șir de caractere

Un șir de caractere poate fi declarat astfel: 

char sir[20];

În acest exemplu sunt alocate 20 de poziții, indexate de la 0 la 19. Deoarece ultimul caracter trebuie să fie '\0', șirul poate conține maximum 19 caractere utile.

La declarare, un șir poate fi inițializat în mai multe moduri: 

char sir[20] = "Salut!";
// primele 6 poziții conțin caracterele 'S', 'a', 'l', 'u', 't', '!',
// iar următoarea poziție conține caracterul '\0'

char t[] = "Salut!";
// dimensiunea tabloului este determinată automat de compilator
// (6 caractere + '\0' = 7 elemente)

char v[6] = {'S', 'a', 'l', 'u', 't', '\0'};
// inițializare caracter cu caracter

char a[] = {'S', 'a', 'l', 'u', 't', '\0'};
// dimensiunea tabloului este determinată automat

Afișarea și citirea unui șir

Afișarea unui șir se poate realiza cu operatorul <<. 

cout << sir;

Citirea unui șir fără spații se poate realiza cu operatorul >>. 

cin >> sir;

Observație: În acest mod pot fi citite doar cuvinte. Citirea se oprește la primul spațiu întâlnit.

Pentru citirea unui șir care conține spații se utilizează funcția cin.getline(). 

cin.getline(sir, lungimea_sirului);
// Se citesc caractere până la apăsarea tastei Enter
// sau până la atingerea lungimii maxime specificate.

Exemplu practic

Programul de mai jos citește numele utilizatorului și numele unui prieten, apoi le afișează pe ecran. 

#include <iostream>
using namespace std;

int main()
{
    char nume1[31], nume2[31];

    cout << "Cum te cheamă? (nume și prenume): ";
    cin.getline(nume1, 31);

    cout << "Cum îl cheamă pe prietenul tău? ";
    cin.getline(nume2, 31);

    cout << "Te numești " << nume1 << endl;
    cout << "Ești prieten cu " << nume2 << endl;

    return 0;
}

Referirea unui caracter din șir. Parcurgerea unui șir de caractere

Șirurile de caractere sunt, de fapt, tablouri unidimensionale de caractere. Fiecare caracter poate fi accesat prin intermediul operatorului [].

Indexarea începe de la 0, astfel încât primul caracter se află pe poziția 0, al doilea pe poziția 1 și așa mai departe. 

char sir[20] = "Salut!";

cout << sir[0]; // afișează 'S'
cout << sir[1]; // afișează 'a'

sir[1] = 'b';   // șirul devine "Sblut!"

cout << sir;    // afișează Sblut!

Funcții pentru șiruri de caractere

Majoritatea funcțiilor pentru șiruri de caractere se găsesc în biblioteca <cstring>.

Determinarea lungimii unui șir - strlen()

Funcția strlen(const char* str) returnează numărul de caractere din șir, fără a lua în considerare caracterul '\0'. 

cout << strlen("Salut!"); // 6

char sir[10] = "Acasa";

cout << strlen(sir);      // 5
cout << strlen(sir + 2);  // 3

Copierea unui șir - strcpy()

Funcția strcpy(char* dest, const char* src) copiază conținutul șirului src în șirul dest.

Primul parametru reprezintă șirul destinație, iar al doilea reprezintă șirul sursă. 

char sir[21], t[21] = "Salut!";

strcpy(sir, "Acasa!");
cout << sir; // Acasa!

strcpy(sir, t);
cout << sir; // Salut!

strcpy(sir, t + 2);
cout << sir; // lut!

strcpy(sir + 2, t);
cout << sir; // luSalut!

Concatenarea șirurilor - strcat()

Funcția strcat(char* dest, const char* src) adaugă șirul src la sfârșitul șirului dest.

Șirul destinație trebuie să aibă suficient spațiu liber pentru a memora rezultatul concatenării. 

char sir[21] = "Salut!";
char t[21] = "Acasa!";

strcat(sir, t);
cout << sir; // Salut!Acasa!

strcat(sir, t + 2);
cout << sir; // Salut!Acasa!asa!

Căutarea unui caracter - strchr()

Funcția strchr(char* str, char ch) caută prima apariție a caracterului ch în șirul str.

Dacă acesta există, funcția returnează un pointer către prima apariție. În caz contrar, returnează NULL. 

char sir[21] = "Salut!";

char *p = strchr(sir, 'a');

cout << p; // alut!

char ch = 'i';

if (strchr("aeiou", ch) != NULL)
{
    cout << "DA";
}
else
{
    cout << "NU";
}

// Se va afișa DA.

Căutarea unui șir într-un alt șir - strstr()

Funcția strstr(char* str, char* str2) caută prima apariție a șirului str2 în șirul str.

Dacă șirul este găsit, funcția returnează un pointer către prima apariție. În caz contrar, returnează NULL. 

char sir[21] = "Salut!";

char *p = strstr(sir, "alu");

cout << p; // alut!

Observație: Pentru orice șir S, apelul strstr(S, "") returnează întotdeauna adresa de început a șirului S.

Compararea șirurilor - strcmp()

Funcția strcmp(char* s, char* t) compară lexicografic două șiruri de caractere.

  • Returnează 0 dacă șirurile sunt identice.
  • Returnează o valoare negativă dacă primul șir este lexicografic mai mic.
  • Returnează o valoare pozitivă dacă primul șir este lexicografic mai mare.
char s1[20] = "ana";
char s2[20] = "ana";
char s3[20] = "andrei";

cout << strcmp(s1, s2); // 0
cout << strcmp(s1, s3); // negativ
cout << strcmp(s3, s1); // pozitiv

Funcția strtok()

Funcția strtok(char* str, const char* sep) împarte un șir în subșiruri (tokeni), folosind caracterele din sep drept delimitatori.

La primul apel se transmite șirul inițial, iar la următoarele apeluri se transmite NULL.

Important: Funcția modifică șirul original, înlocuind delimitatorii cu caracterul '\0'. 

#include <cstring>
#include <iostream>
using namespace std;

int main()
{
    char s[100] = "ana are mere";

    char *p = strtok(s, " ");

    while (p != NULL)
    {
        cout << p << endl;
        p = strtok(NULL, " ");
    }

    return 0;
}

Eliminarea și inserarea unui caracter într-un șir

Eliminarea unui caracter

Pentru eliminarea caracterului aflat pe poziția x, toate caracterele situate după acesta sunt deplasate cu o poziție spre stânga. 

char s[20] = "informatica";
int x = 3;

for (int i = x; i < strlen(s); i++)
    s[i] = s[i + 1];

cout << s;

Inserarea unui caracter

Pentru inserarea unui caracter pe poziția x, toate caracterele începând de la acea poziție sunt deplasate cu o poziție spre dreapta, apoi se inserează caracterul dorit. 

char s[30] = "infoatica";
int x = 4;

for (int i = strlen(s); i >= x; i--)
    s[i + 1] = s[i];

s[x] = 'r';

cout << s;
