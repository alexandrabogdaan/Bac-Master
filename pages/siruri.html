Siruri de caractere

Noțiuni fundamentale despre siruri de caractere în programare

Despre Siruri de Caractere

Vom discuta despre sirurile de caractere reprezentate ca tablouri unidimensionale de caractere în C++. Un sir de caractere este o secvență de caractere stocată într-un tablou de tip char, terminată cu caracterul nul '\0'.
'\0' este un caracter special care indică sfârșitul sirului, reprezentat cu codul ASCII 0 si numit caracter null.

Numele variabilei rămâne constant, dar valoarea se poate schimba.

Declararea unui șir de caractere

Un sir de caractere poate fi declarat astfel: 

            char sir[20];


Acest sir de charactere poate stoca pana la 20 de caractere de la 0-19 (index), dupa ultimul caracter fiind memorat caracteru; '\0'.



La declararea unui sir, acesta poate fi initializat, precum:

            char sir[20] = "Salut!"; // primele 6 caractere sunt 'S', 'a', 'l', 'u', 't', '!', iar al 7-lea este '\0'

            char t[] = "Salut!"; // dimensiunea sirului este determinata automat de compilator, in acest caz 7 (6 caractere + '\0')

            char v[6]={ 'S', 'a', 'l', 'u', 't', '\0' }; // initializare caracter cu caracter

            char a[]={'S', 'a', 'l', 'u', 't', '\0'}; //se aloca automat 7 caractere pentru sir

Afisarea si citirea unui sir

Afisarea se poate face cu operatorul '<<' 

            cout << sir ;


Citirea se poate face cu operatorul '>>' 

            cin >> sir;

!OBSERVATIE, in acest mod, nu se pot citi șiruri de caractere care conțin spații.


Pentru a citi un sir de caractere care contine spatii, se poate folosi functia cin.getline(): 

            getline (char* sir, int lungimea_sirului); //Se vor citi caractere pana la intalnirea lui '\n' sau pana cand se atinge lungimea maxima

Exemplu practic:

Scriem un program C++; 

#include 
using namespace std;
int main()
    {
    char nume1[31], nume2[31];

    cout << "Cum te cheama? (nume, prenume) ";
    cin.getline(nume1, 31);

    cout << "Cum il cheama pe prietenul tau? ";
    cin.getline(nume2 , 31);

    cout << "Te numesti " << nume1 << endl; //afiseaza numele introdus de utilizator
    cout << "Esti prieten cu " << nume2 << endl; //afiseaza numele prietenului
    return 0;
    }

Referirea unui caracter din șir. Parcurgerea unui șir de caractere

Sirurile de caractere sunt, de fapt, tablouri unidimensionale de caractere. Referirea unui caracter din sir se face cu operatorul '[]'. 

            char sir[20] = "Salut!";
            cout << sir[0]; //afiseaza 'S'
            cout << sir[1]; //afiseaza 'a'
            sir[1]='b'; //modifica al doilea caracter din sir, astfel sirul devine "Sblut!"
            cout<< sir; //afiseaza "Sblut!"

Functii cu siruri de caractere

Determinarea lungii unui sir: strlen(const char* str) 

            cout << strlen("Salut!"); // 6
            char sir[10]="Acasa";
            cout << strlen(sir); // 5
            cout << strlen(sir + 2); //3


Copierea unui sir din alt sir: strcpy(char* dest, const char* src) mai intai sirul destinatie, apoi cel sursa. 

            char sir[21], t[21] = "Salut!";
            strcpy(sir , "Acasa!");
            cout << sir; // Acasa!

            strcpy(sir , t);
            cout << sir; // Salut!

            strcpy(sir , t + 2);
            cout << sir; // lut!

            strcpy(sir + 2 , t);
            cout << sir; // luSalut!


Concatenarea a doua siruri: strcat(char* dest, const char* src)

            char sir[21]="Salut!", t[21] = "Acasa!";
            strcat(sir , t);
            cout << sir; // Salut!Acasa!

            strcat(sir , t + 2);
            cout << sir; // Salut!Acasa!asa!


Cautarea unui caracter pana la prima aparitie char *strchr( char * str, char ch );


Functia returneaza NULL daca nu apare caracterul cautat in sir

            char sir[21]="Salut!";
            char *p = strchr(sir , 'a');
            cout << p; // alut!    



            char ch = 'i';
            if(strchr("aeiou" , ch) != NULL)
            {
                cout << "DA";
            }
            else
            {
                cout << "NU";
            }
                //se va afisa DA


Cautarea unui sir in alt sir pana la prima aparitie char *strstr( char * str, char * str2 );

            char sir[21]="Salut!";
            char *p = strstr(sir , "alu");
            cout << p; // alut!

OBSERVATIE! Pentru orice șir S, apelul strstr(S, "") returnează întotdeauna pointerul către începutul șirului S (adică valoarea lui S).



Compararea lexicografica a doua siruri se face cu int strcmp( char * s, char * t );

  • Returneaza 0 daca sirurile sunt egale
  • Returneaza un numar negativ daca primul sir este mai mic decat al doilea
  • Returneaza un numar pozitiv daca primul sir este mai mare decat al doilea
            char sir[21]="abur", t[21]="abecedar";
            if(strcmp(sir , t) < 0)
            {
                cout << "Da";
            }
            else
            {
                cout << "Nu";
            }
                // se va afisa Nu; cuvantul "abur" este lexicografic dupa "abecedar"


Extragerea dintr-un sir de caractere câte un subșir (cuvânt) delimitat de caractere din șirul sep se face astfel: char *strtok( char *str, const char *sep );

            char sep[]=" .,";
            char * p = strtok(sir , sep); //primul apel are ca parametri șirul din care se face extragerea și șirul separatorilor
            while(p != NULL)
            {
                cout << p << endl;
                p = strtok(NULL , sep); //la următoarele apeluri primul parametru este NULL.
            }

            //Secvența de mai jos extrage dintr-un șir sir cuvintele (separate prin caractere din mulțimea {' ', ',', '.'}

Eliminarea si Inserarea unui caracter in siruri

Eliminarea: 

            char sir[256], t[256];
            int x; //pozitia caracterului care se doreste a fi eliminat
            // ...
            //eliminarea
            strcpy(t , sir + x + 1);
            strcpy(sir + x , t);


Inserarea: 

            char sir[256], t[256];
            int x;
            // ...
            //inserarea
            strcpy(t , sir + x);
            strcpy(sir + x + 1 , t);
            sir[x] = 'A'; // echivalent, *(sir+x) = 'A';
