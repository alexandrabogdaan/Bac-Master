Tablouri Bidimensionale

Matrice: declarare, parcurgere și algoritmii clasici de la BAC

Ce este un tablou bidimensional?

Un tablou bidimensional (numit și matrice) este, conceptual, un vector de vectori: o structură organizată pe linii și coloane, în care fiecare element este identificat prin doi indici – indicele liniei (i) și indicele coloanei (j).

a[0][0]
a[0][1]
a[0][2]
a[0][3]
a[1][0]
a[1][1]
a[1][2]
a[1][3]
a[2][0]
a[2][1]
a[2][2]
a[2][3]

O matrice cu n linii și m coloane se declară astfel în C++: 

int a[10][10]; // matrice cu maximum 10 linii si 10 coloane
int n, m;      // numarul real de linii / coloane folosite

// initializare directa
int b[2][3] = { {1, 2, 3}, {4, 5, 6} };
Observație: La fel ca la tablourile unidimensionale, indexarea începe de la 0. O matrice cu n linii și m coloane are linii indexate 0...n-1 și coloane indexate 0...m-1.

Citirea și afișarea unei matrice

Pentru a citi sau afișa toate elementele unei matrice, folosim două bucle imbricate: bucla exterioară parcurge liniile, iar bucla interioară parcurge coloanele din linia curentă. 

int a[10][10], n, m;

cin >> n >> m;

// Citire
for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        cin >> a[i][j];

// Afisare (sub forma de matrice, cu linii si coloane)
for (int i = 0; i < n; i++)
{
    for (int j = 0; j < m; j++)
        cout << a[i][j] << " ";
    cout << endl; // trecem pe linia urmatoare
}

Diagonala principală

Conține elementele a[i][j] pentru care i == j (marcate albastru mai sus).

Diagonala secundară

Conține elementele a[i][j] pentru care i + j == n - 1 (marcate roșu mai sus). 

// Suma elementelor de pe diagonala principala
int sumaPrincipala(int a[10][10], int n)
{
    int s = 0;
    for (int i = 0; i < n; i++)
        s += a[i][i];
    return s;
}

// Suma elementelor de pe diagonala secundara
int sumaSecundara(int a[10][10], int n)
{
    int s = 0;
    for (int i = 0; i < n; i++)
        s += a[i][n - 1 - i];
    return s;
}

Matrice simetrică

Definiție

O matrice pătratică este simetrică dacă elementele simetrice față de diagonala principală sunt egale, adică a[i][j] == a[j][i] pentru orice i, j. 

bool esteSimetrica(int a[10][10], int n)
{
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            if (a[i][j] != a[j][i])
                return false;

    return true;
}
Optimizare: Este suficient să comparăm doar elementele situate peste diagonala principală cu cele simetrice lor, evitând verificările duble (j parcurge doar de la i+1): 
bool esteSimetricaOptim(int a[10][10], int n)
{
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++)
            if (a[i][j] != a[j][i])
                return false;

    return true;
}

Transpusa unei matrice

Transpusa unei matrice A (notată AT) se obține interschimbând liniile cu coloanele: elementul de pe poziția (i, j) din A devine elementul de pe poziția (j, i) în transpusă. 

void transpusa(int a[10][10], int t[10][10], int n, int m)
{
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            t[j][i] = a[i][j];

    // t va avea m linii si n coloane
}
Observație: Dacă matricea originală are n linii și m coloane, transpusa va avea m linii și n coloane – dimensiunile se inversează.

Înmulțirea a două matrice

Două matrice A (n × m) și B (m × p) pot fi înmulțite doar dacă numărul de coloane al lui A este egal cu numărul de linii al lui B. Rezultatul este o matrice C de dimensiune n × p, unde fiecare element se calculează ca suma produselor dintre linia corespunzătoare din A și coloana corespunzătoare din B.

Formula:
c[i][j] = ∑ a[i][k] * b[k][j], pentru k = 0...m-1 
void inmultire(int a[10][10], int b[10][10], int c[10][10], int n, int m, int p)
{
    for (int i = 0; i < n; i++)
    {
        for (int j = 0; j < p; j++)
        {
            c[i][j] = 0;
            for (int k = 0; k < m; k++)
                c[i][j] += a[i][k] * b[k][j];
        }
    }
}

Sume pe linii, coloane și elemente speciale

// Suma elementelor de pe linia i
int sumaLinie(int a[10][10], int m, int i)
{
    int s = 0;
    for (int j = 0; j < m; j++)
        s += a[i][j];
    return s;
}

// Suma elementelor de pe coloana j
int sumaColoana(int a[10][10], int n, int j)
{
    int s = 0;
    for (int i = 0; i < n; i++)
        s += a[i][j];
    return s;
}

// Determinarea elementului maxim din matrice si pozitia sa
void maximMatrice(int a[10][10], int n, int m)
{
    int maxim = a[0][0], li = 0, co = 0;

    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            if (a[i][j] > maxim)
            {
                maxim = a[i][j];
                li = i;
                co = j;
            }

    cout << "Maxim: " << maxim << " pe linia " << li << ", coloana " << co;
}

Rezumat rapid

Operație Condiție pe dimensiuni Complexitate
Parcurgere completă (citire/afișare) oricare n, m O(n × m)
Diagonale (principală/secundară) matrice pătratică (n = m) O(n)
Verificare matrice simetrică matrice pătratică (n = m) O(n2)
Transpusă oricare n, m O(n × m)
Înmulțire matrice coloane(A) = linii(B) O(n × m × p)
