Stiva și Coada

Structuri de date liniare fundamentale: principii, operații și implementare în C++

Ce sunt structurile liniare de date?

Stiva și Coada sunt două structuri de date liniare, în care elementele sunt organizate într-o anumită ordine, dar accesul la ele este restricționat: nu putem accesa direct un element din mijlocul structurii, ci doar prin extremitățile ei, respectând o regulă fixă de inserare și extragere.

Diferența esențială dintre cele două structuri este ordinea în care elementele sunt extrase: la Stivă, ultimul element introdus este primul extras, iar la Coadă, primul element introdus este primul extras.

Stiva (Stack)

Imaginează-ți o stivă de farfurii. Poți pune o farfurie nouă doar pe deasupra stivei, și poți lua tot doar farfuria de pe deasupra – nu poți scoate o farfurie din mijloc fără să dai jos toate cele de pe ea.

Stiva este o structură de tip LIFO (Last In, First Out – ultimul intrat, primul scos). Elementul adăugat cel mai recent este mereu primul care va fi eliminat.

10
25
7
99 ←
vârful stivei (TOP) – aici se introduce și se elimină

Operațiile fundamentale

push(x)

Adaugă elementul x în vârful stivei. Devine noul element accesibil.

pop()

Elimină și returnează elementul din vârful stivei (ultimul introdus).

top() / peek()

Returnează elementul din vârf, fără a-l elimina din stivă.

empty()

Verifică dacă stiva este vidă (nu mai are niciun element).

Implementare în C++ (cu <stack>)

#include <iostream>
#include <stack>
using namespace std;

int main()
{
    stack<int> s;

    s.push(10);
    s.push(25);
    s.push(7);

    cout << s.top(); // afișează 7 (ultimul introdus)

    s.pop();         // se elimină 7

    cout << s.top(); // afișează 25

    cout << s.empty(); // 0 (stiva NU este vidă)
    cout << s.size();  // 2 (numărul de elemente rămase)

    return 0;
}

Implementare manuală (cu vector/tablou)

La BAC, o stivă poate fi simulată simplu cu un tablou și o variabilă care reține poziția vârfului: 

int st[100], varf = -1; // varf = -1 inseamna stiva vida

void push(int x)
{
    varf++;
    st[varf] = x;
}

int pop()
{
    int x = st[varf];
    varf--;
    return x;
}

bool empty()
{
    return varf == -1;
}

Coada (Queue)

Imaginează-ți o coadă la magazin. Persoana care a venit prima este servită prima, iar cei noi se așază mereu la capătul cozii, nu pot trece în față.

Coada este o structură de tip FIFO (First In, First Out – primul intrat, primul scos). Elementele sunt extrase exact în ordinea în care au fost introduse.

10 ←
25
7
99
capul cozii (FRONT) – aici se elimină; la celălalt capăt (REAR) se introduce

Operațiile fundamentale

push(x) / enqueue(x)

Adaugă elementul x la coada structurii (capătul REAR).

pop() / dequeue()

Elimină și returnează elementul din capul cozii (capătul FRONT), cel mai vechi introdus.

front()

Returnează elementul din capul cozii, fără a-l elimina.

empty()

Verifică dacă coada este vidă (nu mai are niciun element).

Implementare în C++ (cu <queue>)

#include <iostream>
#include <queue>
using namespace std;

int main()
{
    queue<int> q;

    q.push(10);
    q.push(25);
    q.push(7);

    cout << q.front(); // afișează 10 (primul introdus)

    q.pop();           // se elimină 10

    cout << q.front(); // afișează 25

    cout << q.empty(); // 0 (coada NU este vidă)
    cout << q.size();  // 2 (numărul de elemente rămase)

    return 0;
}

Implementare manuală (cu vector/tablou circular)

La BAC, o coadă simplă poate fi simulată cu un tablou și două variabile, prim și ultim: 

int c[100], prim = 0, ultim = -1; 
// coada vida: prim > ultim

void push(int x)
{
    ultim++;
    c[ultim] = x;
}

int pop()
{
    int x = c[prim];
    prim++;
    return x;
}

bool empty()
{
    return prim > ultim;
}

Rezumat rapid: Stivă vs. Coadă

Criteriu Stivă (Stack) Coadă (Queue)
Principiu LIFO – ultimul intrat, primul scos FIFO – primul intrat, primul scos
Capete de lucru Un singur capăt (vârful / TOP), folosit atât pentru inserare cât și pentru extragere. Două capete: unul pentru inserare (REAR), altul pentru extragere (FRONT).
Operații de bază push(), pop(), top(), empty() push()/enqueue(), pop()/dequeue(), front(), empty()
Header C++ <stack> <queue>
Analogie Stivă de farfurii / cărți Coadă la magazin / casierie

Aplicații practice frecvente la BAC

Aplicații ale Stivei

  • Verificarea corectitudinii parantezelor într-o expresie (paranteze echilibrate).
  • Evaluarea expresiilor aritmetice (notație poloneză / postfixată).
  • Implementarea funcției recursivitate (Stiva de Apeluri – Call Stack).
  • Funcția "Undo" dintr-un editor de text sau parcurgerea istoricului unui browser.

Aplicații ale Cozii

  • Gestionarea proceselor într-un sistem de operare (rând de așteptare la procesor).
  • Parcurgerea în lățime a unui graf (BFS – Breadth-First Search).
  • Sisteme de tip "primul venit, primul servit" (cozi la imprimantă, ghișeu etc.).
  • Simularea unui buffer de date care se transmit în ordinea sosirii.

Concluzie pentru examen

Regula de aur la BAC:
Citește mereu cu atenție ordinea operațiilor din enunț. Dacă ultimul element introdus este primul cerut/extras → ai de-a face cu o Stivă (LIFO). Dacă primul element introdus este primul cerut/extras → ai de-a face cu o Coadă (FIFO).
