Operatori în C++

Tipurile de operatori, ordinea priorităților și capcanele frecvente de la BAC

Ce este un operator?

Un operator este un simbol care indică o operație (de calcul, de comparare, de atribuire etc.) ce trebuie efectuată asupra unuia sau mai multor operanzi. În funcție de numărul operanzilor, operatorii pot fi unari (un singur operand), binari (doi operanzi) sau ternari (trei operanzi).

Operatori aritmetici

Se folosesc pentru efectuarea calculelor matematice de bază.

Operator Semnificație Exemplu
+Adunare5 + 3 → 8
-Scădere5 - 3 → 2
*Înmulțire5 * 3 → 15
/Împărțire7 / 2 → 3 (între întregi)
%Restul împărțirii (modulo)7 % 2 → 1
Capcană frecventă: Operatorul / are comportament diferit în funcție de tipul operanzilor! Între două numere întregi, rezultatul este trunchiat (partea fracționară se elimină): 7 / 2 → 3, nu 3.5. Pentru rezultat cu zecimale, cel puțin un operand trebuie să fie float/double: 7.0 / 2 → 3.5.

Operatorul % (modulo) se aplică doar numerelor întregi și este folosit frecvent pentru a testa paritatea unui număr:

if (n % 2 == 0)
    cout << "n este par";
else
    cout << "n este impar";

Operatori de atribuire

Operatorii de atribuire combinată sunt prescurtări utile, des folosite la BAC pentru a "păcăli" la nivel de citire a codului.

Operator Echivalent cu Exemplu (x = 10)
=atribuire simplăx = 5; → x devine 5
+=x = x + valx += 3; → x devine 13
-=x = x - valx -= 3; → x devine 7
*=x = x * valx *= 2; → x devine 20
/=x = x / valx /= 2; → x devine 5
%=x = x % valx %= 3; → x devine 1

Operatori de incrementare/decrementare

Operatorii ++ și -- măresc, respectiv micșorează valoarea unei variabile cu 1. Poziția lor (înainte sau după variabilă) schimbă momentul în care modificarea are efect.

Preincrementare: ++x

Valoarea lui x este mărită înainte de a fi folosită în expresie.

Postincrementare: x++

Valoarea lui x este folosită întâi în expresie, apoi este mărită.

int x = 5;

cout << ++x; // afișează 6 (x devine 6, apoi se afișează)
cout << x;   // afișează 6

int y = 5;

cout << y++; // afișează 5 (se afișează 5, apoi y devine 6)
cout << y;   // afișează 6
Observație: Diferența dintre ++x și x++ este vizibilă doar atunci când operatorul este folosit într-o expresie (de exemplu, în cout sau ca parametru). Folosit izolat, pe o linie separată, efectul final este identic.

Operatori relaționali și logici

Sunt folosiți pentru a construi condiții (expresii cu valoare de adevăr: true/false, reprezentate intern prin 1/0).

Operatori relaționali

  • == – egal
  • != – diferit
  • < – mai mic
  • > – mai mare
  • <= – mai mic sau egal
  • >= – mai mare sau egal

Operatori logici

  • && – ȘI (AND): adevărat doar dacă ambele condiții sunt adevărate
  • || – SAU (OR): adevărat dacă cel puțin una dintre condiții este adevărată
  • ! – NEGAȚIE (NOT): inversează valoarea de adevăr
int n = 7;

if (n > 0 && n % 2 == 1)
    cout << "n este pozitiv si impar";

if (n < 0 || n == 0)
    cout << "n este negativ sau nul";

cout << !(n > 10); // 1 (adevarat), pentru ca n > 10 este fals
Capcană frecventă: Nu confunda operatorul de egalitate == cu operatorul de atribuire =! Scrierea if (n = 5) în loc de if (n == 5) este o greșeală clasică, care atribuie valoarea 5 variabilei n, în loc să o compare.

Ordinea (prioritatea) operatorilor

Atunci când o expresie conține mai mulți operatori, evaluarea respectă o ordine de priorități, similară cu cea din matematică. De la cea mai mare prioritate (se evaluează primul) la cea mai mică:

Nivel Operatori Tip
1 () Paranteze (cea mai mare prioritate)
2 ++ -- ! Unari (incrementare, negație)
3 * / % Înmulțire, împărțire, rest
4 + - Adunare, scădere
5 < <= > >= Relaționali de ordine
6 == != Relaționali de (in)egalitate
7 && ȘI logic
8 || SAU logic
9 = += -= ... Atribuire (cea mai mică prioritate)
Observație: Operatorii *, /, % au prioritate mai mare decât + și -, exact ca în matematică. Folosirea parantezelor este cea mai sigură metodă de a evita confuziile la evaluarea expresiilor complexe. 
int rezultat = 2 + 3 * 4;     // 14, nu 20 (inmultirea se face prima)
int rezultat2 = (2 + 3) * 4;  // 20, parantezele schimba ordinea

bool test = 5 > 3 && 2 == 2;  // se evalueaza intai 5 > 3 si 2 == 2, apoi &&

Operatorul ternar ?:

Operatorul ternar este singurul operator din C++ cu trei operanzi. Este o variantă compactă a structurii if-else, utilă atunci când dorim să atribuim o valoare în funcție de o condiție, pe o singură linie.

Sintaxă:
condiție ? valoare_daca_adevarat : valoare_daca_fals 
int a = 7, b = 12;

int maxim = (a > b) ? a : b;
cout << maxim; // 12

// echivalent cu:
int maxim2;
if (a > b)
    maxim2 = a;
else
    maxim2 = b;

Operatorul sizeof

Operatorul sizeof returnează numărul de octeți (bytes) ocupați în memorie de un tip de date sau de o variabilă. 

cout << sizeof(int);    // de regula 4 (octeti)
cout << sizeof(char);   // 1
cout << sizeof(double); // de regula 8

int v[10];
cout << sizeof(v);      // 40 (10 elemente x 4 octeti)
