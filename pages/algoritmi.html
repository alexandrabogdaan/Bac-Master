Probleme care operează asupra cifrelor unui număr

Prelucrarea cifrelor se bazează pe descompunerea matematică în baza 10. Izolarea ultimei cifre se realizează prin operatorul modulo (n%10), în timp ce eliminarea fizică a acesteia din număr se face prin împărțire întreagă (n/10). Procesul se repetă într-o structură repetitivă cât timp numărul este diferit de 0.

Oglinditul (Inversul)

Construiește inversul cifrelor prin deplasarea succesivă a rezultatului spre stânga (înmulțire cu 10) și adunarea ultimei cifre curente.

int n, ogl=0;
while (n!=0)
{
    ogl=ogl*10+n%10;
    n/=10;
}

Suma și Numărarea

Iterează prin număr, adunând valoarea fiecărei cifre în variabila sumă și incrementând un contor dedicat pentru fiecare pas executat.

int n, s=0, cn=0;
while (n!=0)
{
    s+=n%10;
    cn++;
    n/=10;
}

Divizibilitate. Numere prime. Algoritmul lui Euclid

Verificarea proprietății de număr prim

Eficientizarea algoritmului constă în limitarea căutării divizorilor proprii până la valoarea părții întregi a rădăcinii pătrate a numărului (d*d<=n). Dacă nu se identifică niciun divizor în acest interval, numărul este prim.

Algoritmul lui Euclid (CMMDC)

Află cel mai mare divizor comun utilizând operația de obținere a restului împărțirilor succesive, metodă superioară scăderilor repetate.

int a, b, r;
while (b!=0)
{
    r=a%b;
    a=b;
    b=r;
}

Cel mai mic multiplu comun (CMMMC)

Se calculează pe baza relației matematice directe dintre produsul celor două numere și cel mai mare divizor comun al acestora.

int a, b, cmmmc, copie_a, copie_b, r;
copie_a=a;
copie_b=b;
while (b!=0)
{
    r=a%b;
    a=b;
    b=r;
}
cmmmc=(copie_a*copie_b)/a;

Şirul lui Fibonacci. Calculul unor sume

Șirul lui Fibonacci este o succesiune numerică în care primele două elemente sunt egale cu 1, iar fiecare termen următor este generat prin însumarea directă a celor doi termeni strict precedenți. Calculul sumelor cu termen general implică generarea riguroasă a fiecărui component la fiecare pas al buclei.

Generare Șirul lui Fibonacci

Folosește trei variabile pentru a păstra cei doi termeni anteriori și valoarea curentă calculată, reînoind valorile la fiecare pas.

int a=1, b=1, c, i, n;
cout<<a<<" "<<b<<" ";
for (i=3; i<=n; i++)
{
    c=a+b;
    cout<<c<<" ";
    a=b;
    b=c;
}

Calcul Sume (Exemplu: Termen Factorial)

Pentru a evita recalcularea costisitoare, numitorul (factorialul) se construiește incremental, prin înmulțire cu indicele curent.

double s=0, p=1;
int i, n;
for (i=1; i<=n; i++)
{
    p*=i;
    s+=1.0/p;
}

Determinare minim/maxim

Algoritmul presupune inițializarea ipotetică a valorii extreme cu primul element al structurii liniare. Ulterior, vectorul este parcurs integral, comparând fiecare element cu valoarea de referință și actualizând-o ori de câte ori este identificată o valoare mai mare (pentru maxim) sau mai mică (pentru minim).

int maxim=v[0], i, n;
for (i=1; i<n; i++)
{
    if (v[i]>maxim)
    {
        maxim=v[i];
    }
}

Metode de ordonare

Sortarea reprezintă procesul de rearanjare a elementelor unui vector într-o structură monotonă (crescătoare sau descrescătoare). Alegerea metodei corecte depinde de restricțiile de timp și de structura datelor de intrare.

Vizualizare Pas cu Pas: Metoda Bulelor

Apăsați pe pornire pentru a urmări evoluția elementelor în memorie.

Sursa C++: Metoda Bulelor (Bubble Sort)

Parcurge vectorul în mod repetat și compară elementele vecine. Dacă acestea se află în ordine inversă, le interschimbă valorile. La fiecare iterație completă, cel mai mare element nerezolvat migrează spre poziția sa finală la extremitatea dreaptă.

int i, j, aux, n, v[100];
for (i=0; i<n-1; i++)
{
    for (j=0; j<n-1-i; j++)
    {
        if (v[j]>v[j+1])
        {
            aux=v[j];
            v[j]=v[j+1];
            v[j+1]=aux;
        }
    }
}
Metodă Descriere Teoretică și Implementare C++
Metoda Inserției

Consideră că primele elemente sunt deja sortate. Ia fiecare element nou și îl inserează în poziția corectă printre cele deja verificate, deplasând restul elementelor spre dreapta.

int i, j, x, n, v[100];
for (i=1; i<n; i++)
{
    x=v[i];
    j=i-1;
    while (j>=0 && v[j]>x)
    {
        v[j+1]=v[j];
        j--;
    }
    v[j+1]=x;
}
Metoda Selecției

Compară fiecare element de pe o poziție fixată cu toate elementele aflate după el. Dacă se găsește o valoare mai mică, pozițiile sunt inversate imediat.

int i, j, aux, n, v[100];
for (i=0; i<n-1; i++)
{
    for (j=i+1; j<n; j++)
    {
        if (v[i]>v[j])
        {
            aux=v[i];
            v[i]=v[j];
            v[j]=aux;
        }
    }
}
Metoda Numărării

Determină poziția finală a fiecărui element prin numărarea valorilor din vector care sunt strict mai mici decât el, plasând elementul direct la indexul rezultat într-un vector suplimentar.

int i, j, rez, n, v[100], b[100];
for (i=0; i<n; i++)
{
    rez=0;
    for (j=0; j<n; j++)
    {
        if (v[j]<v[i] || (v[j]==v[i] && j<i))
        {
            rez++;
        }
    }
    b[rez]=v[i];
}

Interclasare

Interclasarea reprezintă tehnica de combinare a doi vectori ordonați în mod nativ într-un al treilea vector, păstrând proprietatea de sortare. Procesul folosește indici independenți pentru parcurgerea structurilor paralele și este mult mai eficient decât concatenarea urmată de o sortare clasică, având complexitate liniară.

int i=0, j=0, k=0, n, m, A[100], B[100], C[200];
while (i<n && j<m)
{
    if (A[i]<B[j]) 
    {
        C[k++]=A[i++];
    }
    else 
    {
        C[k++]=B[j++];
    }
}
while (i<n) 
{
    C[k++]=A[i++];
}
while (j<m) 
{
    C[k++]=B[j++];
}

Metode de căutare

Vizualizare Pas cu Pas: Căutare Binară

Introduceți valoarea căutată și porniți simularea. Vectorul este sortat automat în prealabil.

Căutare Secvențială (Liniară)

Potrivită pentru structuri nesortate. Verifică consecutiv fiecare element în parte, de la stânga la dreapta, până când valoarea căutată este identificată în tablou sau vectorul se epuizează.

int poz=-1, i, n, x, v[100];
for (i=0; i<n; i++)
{
    if (v[i]==x)
    {
        poz=i;
        break;
    }
}

Căutare Binară

Necesită obligatoriu un vector sortat. Algoritmul compară valoarea căutată cu mijlocul intervalului curent, înjumătățind spațiul de căutare la fiecare pas nefavorabil prin reajustarea limitelor stânga/dreapta.

int st=0, dr=n-1, poz=-1, mij, x, v[100];
while (st<=dr)
{
    mij=(st+dr)/2;
    if (v[mij]==x)
    {
        poz=mij;
        break;
    }
    else if (v[mij]<x) 
    {
        st=mij+1;
    }
    else 
    {
        dr=mij-1;
    }
}
