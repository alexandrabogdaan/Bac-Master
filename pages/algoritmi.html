6.1. Probleme care operează asupra cifrelor unui număr

Extragerea cifrelor unui număr se face repetat prin izolarea ultimei cifre (n % 10) și eliminarea ei (n / 10).

Oglinditul (Inversul)

int ogl = 0;
while (n != 0) {
    ogl = ogl * 10 + n % 10;
    n /= 10;
}

Suma și Numărarea

int s = 0, cn = 0;
while (n != 0) {
    s += n % 10;
    cn++;
    n /= 10;
}

6.3. Şirul lui Fibonacci. Calculul unor sume

Fiecare termen este suma celor doi termeni precedenți: f(n) = f(n-1) + f(n-2).

Generare Șirul lui Fibonacci

int a = 1, b = 1, c;
cout << a << " " << b << " ";
for (int i = 3; i <= n; i++) {
    c = a + b;
    cout << c << " ";
    a = b;
    b = c;
}

Calcul Sumpă cu termen general

double s = 0, p = 1;
for (int i = 1; i <= n; i++) {
    p *= i; // Termen general factorial
    s += 1.0 / p;
}

6.4. Determinare minim/maxim

Determinarea celei mai mici sau mai mari valori dintr-o secvență sau un tablou liniar.

int maxim = v[0];
for (int i = 1; i < n; i++) {
    if (v[i] > maxim) {
        maxim = v[i];
    }
}

6.5. Metode de ordonare

Rearanjarea elementelor unui vector în ordine crescătoare sau descrescătoare.

Vizualizare Pas cu Pas: Metoda Bulelor

Apăsați pe pornire pentru a urmări evoluția elementelor în memorie.
Metodă Implementare C++
Metoda Inserției 
for (int i = 1; i < n; i++) {
    int x = v[i], j = i - 1;
    while (j >= 0 && v[j] > x) {
        v[j + 1] = v[j];
        j--;
    }
    v[j + 1] = x;
}
Metoda Selecției 
for (int i = 0; i < n - 1; i++) {
    for (int j = i + 1; j < n; j++) {
        if (v[i] > v[j]) swap(v[i], v[j]);
    }
}
Metoda Numărării 
for (int i = 0; i < n; i++) {
    int rez = 0;
    for (int j = 0; j < n; j++)
        if (v[j] < v[i] || (v[j] == v[i] && j < i)) rez++;

6.6. Interclasare

Combinarea a doi vectori deja sortați (A cu n elemente și B cu m elemente) într-un al treilea vector sortat C.

int i = 0, j = 0, k = 0;
while (i < n && j < m) {
    if (A[i] < B[j]) C[k++] = A[i++];
    else C[k++] = B[j++];
}
while (i < n) C[k++] = A[i++];
while (j < m) C[k++] = B[j++];

6.7. Metode de căutare

Vizualizare Pas cu Pas: Căutare Binară

Introduceți valoarea căutată și porniți simularea. Vectorul este sortat automat în prealabil.

Căutare Secvențială (Liniară)

Se verifică element cu element. Funcționează pe vectori nesortați.

int poz = -1;
for (int i = 0; i < n; i++) {
    if (v[i] == x) { poz = i; break; }
}

Căutare Binară

Înjumătățește repetat intervalul. Necesită obligatoriu vector sortat.

int st = 0, dr = n - 1, poz = -1;
while (st <= dr) {
    int mij = (st + dr) / 2;
    if (v[mij] == x) { poz = mij; break; }
    else if (v[mij] < x) st = mij + 1;
    else dr = mij - 1;
}
