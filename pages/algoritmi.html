Algoritmi de Bază Completă

Toți algoritmii obligatorii pe numere și tablouri din programa de BAC

1. Interschimbarea a două variabile (Swap)

Folosirea unei variabile auxiliare (paharul al treilea) pentru a schimba valorile între ele.

int aux = a;
a = b;
b = aux;
💡 Truc de BAC: Schimbarea fără auxiliar (folosind operații matematice): a = a + b; b = a - b; a = a - b;

2. Prelucrarea cifrelor unui număr

Spargerea unui număr în cifre folosește operatorii % 10 (pentru a lua ultima cifră) și / 10 (pentru a o tăia).

// Oglinditul (Palindrom) și extragerea cifrelor
int oglindit = 0;
while (n != 0) {
    int cifra = n % 10;
    oglindit = oglindit * 10 + cifra;
    n = n / 10;
}

3. Verificarea unui număr prim

Căutăm divizori pari sau impari doar în intervalul [2, √n].

Simulator Pas cu Pas (Viteză Redusă)

Introduceți un număr și porniți urmărirea.
bool estePrim(int n) {
    if (n < 2) return false;
    for (int d = 2; d * d <= n; d++)
        if (n % d == 0) return false;
    return true;
}

4. Descompunerea în factori primi

Aflarea tuturor numerelor prime care înmulțite dau numărul inițial (foarte cerut la Subiectul II/III).

int d = 2;
while (n > 1) {
    int p = 0;
    while (n % d == 0) {
        p++;
        n /= d;
    }
    if (p > 0) {
        cout << d << "^" << p << endl;
    }
    d++;
}

5. CMMDC (Algoritmul lui Euclid)

while (b != 0) {
    int r = a % b;
    a = b;
    b = r;
}
// CMMDC se află acum în 'a'

6. Bubble Sort (Metoda Bulelor)

Vizualizare Pas cu Pas Completă (Lentă)

Log-ul va genera explicațiile linie cu linie, cu pauze mari între pași.
void bubbleSort(int v[], int n) {
    for (int i = 0; i < n - 1; i++)
        for (let j = 0; j < n - 1 - i; j++)
            if (v[j] > v[j + 1]) {
                int aux = v[j];
                v[j] = v[j + 1];
                v[j + 1] = aux;
            }
}

7. Sortarea prin Selecție Directă

Se caută elementul minim din restul vectorului și se aduce în poziția curentă.

for (int i = 0; i < n - 1; i++) {
    for (int j = i + 1; j < n; j++) {
        if (v[i] > v[j]) {
            int aux = v[i];
            v[i] = v[j];
            v[j] = aux;
        }
    }
}
