Tablouri unidimensionale; Vectori

Noțiuni fundamentale despre declararea și utilizarea vectorilor

Definiția vectorului

Un tablou unidimensional se declară în C++ astfel:

tipDeDate denumire[Dimensiune];

Ex1: int X[10];

Ex2: int n,X[10],m,Y[100];

Observație!

  • Elementele unui tablou declarat global sunt inițializate cu 0! Dacă tabloul este declarat local, valorile sunt inițializate cu valori aleatorii
  • Primul element al unui tablou este indexat de la 0

Referirea unui element

Referirea unui element se face prin indexul său în interiorul tabloului: Referirea unui element se face prin operatorul de indexare, [], care are prioritate maximă


De exemplu:

  • X[0]
  • X[5]
  • X[i]

X este tabloul, iar 0,5,i sunt indicii. Indici trebuie să fie întregi și să se încadreze în dimensiunea tabloului.


Un element al tabloului referit prin indice este tratat ca o variabilă oarecare de tipul stabilit la declarare

        int X[10];
        X[0] = 17;

        cout << X[0]; // 17
        cout << X[0] + 5; // 22
        cout << X[0] / 5; // 3

Dimensiunea unui tablou unidimensional

La declararea unui tablou unidimensional se precizează o dimensiune pentru acesta. Aceasta reprezintă o dimensiune fizică a tabloului – numărul maxim de elemente pe care le-ar putea avea acesta, conform restricțiilor problemei.

Parcurgerea unui tablou unidimensional

Parcurgerea unui tablou reprezintă referirea fiecărui element al tabloului, într-o anumită ordine. Referirea elementului se face prin intermediul indicelui, cu ajutorul operatorului de indexare.


Exemplu:

                int X[100], n;
                    //unde n este numărul de elemente efectiv utilizate în tablou
                for(int i = 0 ; i < n ; i ++) // de la 0 la n-1!! în ordine crescătoare
                    X[i] = 1; //inițializăm toate elementele cu valoarea 1

Citirea unui vector

În majoritatea cazurilor, citirea NU se poate face așa: 

 cin>>X;

Aceasta va da eroare de sintaxă. În schimb, elementele tabloului pot fi citite în felul următor:

                int X[100], n;

                cin >> n;
                for(int i = 0 ; i < n ; i ++)
                cin >> X[i]; //de la consola pot fi atribuite valori de tipul 'int'

Afișarea unui vector

La fel ca la citire, afișarea unui vector NU se poate face așa: 

 cout<< X;

Totuși, acest lucru nu va duce la eroare de sintaxă, ci va afișa adresa de memorie a primului element ex. 0x7ffc9711bcd0


Afișarea corectă:

                for(int i = 0 ; i < n ; i ++)
                cout << X[i] << ' ';

Indexare de la 0 și indexare de la 1

Orice tablou C++ are fizic elementele indexate de la 0 la Dimensiune-1.

De exemplu, dacă avem nevoie de un tablou cu n≤100 elemente întregi, îl vom declara:

int X[100];

iar elementele vor avea indici între 0 și 99. Astfel, primul element al tabloului este X[0], al doilea este X[1], …, ultimul este X[n-1].


Dacă dorim, putem ignora elementul X[0] (pur și simplu nu îl folosim, el există însă în continuare), și atunci toate operațiile (citire, afișare, parcurgere) vor utiliza elementele X[1] (primul element), X[2] (al doilea element), …, X[n] (ultimul element).


Trebuie de asemenea să tratăm cu atenție declararea tabloului, dimensiunea fizică trebuind să fie mai mare cu 1 decât valoarea precizată în problemă. Dacă de exemplu, n≤100, declararea va fi:

int X[101];

Acest lucru este necesar pentru a exista elementul X[100].

Inițializarea elementelor unui vector

La declararea tabloului este posibil să inițializăm elementele sale: 

                int A[] = {10 , 20 , 30 , 40}; 
                int B[10] = {10 , 20 , 30 , 40}; 
                int C[10] = {1};

A va avea 4 elemente, A[0] = 10, A[1] = 20, A[2] = 30, A[3] = 40


B va avea 10 elemente, B[0] = 10, B[1] = 20, B[2] = 30, B[3] = 40, restul elementelor vor fi inițializate cu 0


C va avea 10 elemente, toate inițializate cu 0 ÎNAFĂRĂ DE C[0]
